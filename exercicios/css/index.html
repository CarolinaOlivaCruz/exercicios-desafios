HTML

        
O que é HTML?

        
  HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar
         uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, 
         em uma lista com marcadores ou usando imagens e tabelas.


        
CSS

        
 O que CSS?

        
 CSS (Folha de Estilo em Cascata) é o código que você usa para dar estilo à sua página Web.
